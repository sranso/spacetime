Things To Try

  • Hover over one of the bold "fibonaci"s and hit tab.
  • Hover over the non-bold "fibonaci" and give it a new name by typing "newfib".
  • From anywhere (hovering over any node), press space and type "cool". Hit tab while over that new node to expand it. Now move (drag) any of the nodes under the two identical child nodes (see help for more info).

Help

  • Nodes can appear in multiple places. The dot will show if nodes are the same.
  • Expand/collapse nodes by hovering over them and hitting tab.
  • Move nodes tree-wise by clicking below the dot and dragging.
  • Move nodes "tower"-wise by clicking well below the text and dragging up/down/left/right.
  • Moving nodes "tower"-wise reshapes the tree.
  • Select multiple towers by holding shift while hovering over them.
  • Delete nodes by pressing 3.
  • Copy nodes by pressing 2.
  • Copying nodes by reference (bold) will make another instance of the same node. Editing either node will edit the other one as well.
  • Copying nodes by value (non-bold) will make a different node, but the value will initially be the same until one or the other is edited.
  • Insert a node by hovering over the node to insert after and then pressing space.
  • Edit the text of a node by hovering over it and typing.
  • Editing the text of a reference (bold node), changes what node should appear at that location.
  • Editing the text of a value node (non-bold) changes the text in any location that node appears.
  • Tabbing a value node (non-bold) creates a reference, allowing it to be autocompleted or copied by reference.
  • Autocomplete does a copy by reference.
  • Try '(', ')', '[', or ']' while typing to change the height.
  • Typing ',' is like typing a ')' followed by a '('. It separates two nodes that are at the same height.
  • Press '#' to enter/increment a number, followed by any number keys.
