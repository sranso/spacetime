I demoed two versions of my side project at the Future Programming Workshop in October. They are...

Spacetime (new version)

Spacetime is a programming environment for 2D graphics with a different approach to time. Programs are a two dimensional grid of cells with time as the x dimension.

My original motivation for this project was to explore alternatives to programming via editing text in a file, but lately I've been thinking it would be cool to make games, interactive art, and explorable explanations. A goal is that it should be simple for non-programmers to learn. I'd also like to promote sharing and building off of each other's work by making programs composable and copy/paste friendly.

You can try out Spacetime here. The demo mentions most of the keyboard shortcuts except that "d" deletes a row and "D" deletes a column. Another graphics command is "rotate".

Interested?

Follow the development of Spacetime on the Google group, or follow me on Twitter. See the source and submit issues or pull requests on Github.



Pancakes (older version)

This older version is actually more complete than Spacetime, but the ideas aren't as powerful. Still, a lot of what I did here I'd like to port over to Spacetime.
